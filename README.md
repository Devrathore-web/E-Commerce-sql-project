#  E-Commerce SQL Project

##  Overview
This project is a SQL-based E-Commerce Database System.

##   Tables Used
- Users
- Products
- Orders
- Order_Items
- Payments

##   Key Features
- Managed customer orders and products
- Calculated total revenue
- Identified top customers
- Found most sold products

##  Skills Used
- SQL
- Joins
- Group By
- Aggregate Functions

##  Outcome
Learned how to design and analyze a real-world database system using SQL.
